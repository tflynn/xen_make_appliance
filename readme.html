Introduction

xen_create_appliance is a script to create Xen appliances.

It has only been tested on Centos 5 64-bit build with the standard Xen kernel found in the distribution.

Prerequisites

Dropbox account and setup

The dropbox account is used to hold generated configuration file so that they are accessible during initialization of a VM. Any other mechanism that provides a read/write store could be equally well used.

Install Dropbox for Linux. Probably connected it to a non-VM instance (this is the only combination tested).

This script will create a directory tree in ~/Dropbox/Public/xen with VM-specific subdirectories containing configuration files.

LVM Setup

An LVM setup is assumed. See the separate 'readme_lvm.html'.

Configuration

There are several files which are not part of the distribution (deliberately). These are:

  • dropbox_base_url - A file containing a public URL to a dropbox account
  • xen_create_appliance.config - A file containt site-specific configuration information for this script.

File: dropbox_base_url

DROPBOX_BASE_URL=http://dl.dropbox.com/u/9999999/Public/xen

File: xen_create_appliance.config
